## Wymagania

Projekt wymaga:
Python 3.x
Django


## Instalacja

Zainstaluj wymagane zależności:
    ```
    pip install pipenv
    ```

## Użycie

Aby uruchomić projekt, wykonaj następujące polecenia:
    ```
    pipenv shell
    ```
    ```
    pipenv install django
    ```
    ```
    python manage.py runserver
    ```
